# SnakeOLED_128x64  
Аркадная змейка на Arduino + 0.96" SSD1306 OLED и четырёх кнопках.

## Железо
- Arduino Uno/Nano (ATmega328)  
- OLED 128×64 I²C (SSD1306)  
- 4 кнопки (UP/DOWN/LEFT/RIGHT) → пины 8, 9, 10, 11  
- Пьезо-зуммер → пин 6  

## Библиотеки
- Adafruit_GFX  
- Adafruit_SSD1306  

Установить через Менеджер библиотек Arduino IDE.

## Собираем 
можно сделать плату с помощью текстолита
